class EndpointWrapper[source]¶

Wraps endpoints for client. When an attribute is fetched from this class, a partial version of EndpointWrapper.generic() is returned with the attribute name pre-placed in the method parameter.

This class is not accessed directly, but invoked through an instance: spanclient.handles.

Decorator that is ACTUALLY called decorating an endpoint method.

Parameters

  • method (str) – HTTP method – GET, POST, PUT, etc. Filled in automatically by the decorator invoked, ie: @handles.get

  • endpoint (str) – Endpoint path. Can use f-string syntax for path params. ex: /wizards/{wizard_id}

  • query_params (Optional[MutableMapping[str, Any]]) – URL query params to apply to all requests from this method.

  • headers (Optional[MutableMapping[str, Any]]) – HTTP headers to apply to all requests made from this method.

  • mimetype_send (Union[MimeType, str, None]) – Mimetype to use when encoding content. Added to the 'Content-Type' header.

  • mimetype_accept (Union[MimeType, str, None]) – Mimetype to request from server. Added to the 'Accept' header.

  • req_schema (Optional[Schema]) – Schema for dumping request body media.

  • resp_codes (Union[int, Tuple[int, …]]) – Valid response codes.

  • resp_schema (Optional[Schema]) – Schema for loading response body content.

  • data_updater (Optional[Callable[[~ModelType, Any], None]]) – To use when updating existing data objects in-place.

  • return_info (bool) – Whether to return a ReturnData instance in place of the decoded / loaded response body.

Return type

Raises

  • StatusMismatchError – When response status does not match resp_codes.

classmethod is_mimetype(value, mimetype)[source]¶

Checks if string or enum is equivalent to a given mimetype enum value. Similar to isinstance() or issubclass().

Parameters

  • value (Union[MimeType, str, None]) – to check

  • mimetype (MimeType) – to check against.

Return type

bool

Returns

True: is equivalent. False: is not Equivalent.

Does some cleaning (such as resolving case and removing ‘x-‘ from type).

For instance, all of the following will pass equivalence for MimeType.YAML:

  • ‘application/yaml’

  • ‘application/YAML’

  • ‘application/x-yaml’

  • ‘yaml’

  • ‘x-yaml’

mock_aiohttp(method, resp=None, req_validator=None)[source]¶

Decorator for patching aio_http method with request validator that returns a MockResponse.

Parameters

  • method (str) – HTTP method to replace: GET, POST, etc.

  • resp (Union[MockResponse, List[Optional[MockResponse]], None]) – Mock response object(s) to return. Objects are returned endlessly, so if one mock response is passed, it will be returned every time the aiohttp method is invoked, but if three are passed, they will be rotated through like so: 1, 2, 3, 1, 2, 3, 1, …

  • req_validator (Union[RequestValidator, List[Optional[RequestValidator]], None]) – Validator(s) to check passed aiohttp params. Like mock responses, validators are cycled through endlessly to check each response.

Return type

Callable

Returns

Pytest decorator.
